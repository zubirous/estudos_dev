# Prova: ARQUITETURA CLIENTE-SERVIDOR

## Provas de Alternativas

## Questão 1 (Múltipla Escolha)
Na arquitetura Cliente-Servidor, o que o cliente faz?

a) Processa requisições
b) Faz requisições ao servidor
c) Armazena dados
d) Não faz nada

**Resposta:** b) Faz requisições ao servidor

---

## Questão 2 (Múltipla Escolha)
Na arquitetura Cliente-Servidor, o que o servidor faz?

a) Faz requisições
b) Processa requisições e retorna respostas
c) Armazena dados apenas
d) Não faz nada

**Resposta:** b) Processa requisições e retorna respostas

---

## Questão 3 (Múltipla Escolha)
Qual é a diferença entre 2-Tier e 3-Tier?

a) Não há diferença
b) 2-Tier: Cliente ↔ Servidor, 3-Tier: Cliente ↔ Servidor de Aplicação ↔ Servidor de BD
c) 2-Tier tem mais camadas
d) 3-Tier não tem servidor

**Resposta:** b) 2-Tier: Cliente ↔ Servidor, 3-Tier: Cliente ↔ Servidor de Aplicação ↔ Servidor de BD

---

## Questão 4 (Múltipla Escolha)
Na arquitetura 2-Tier:

a) Cliente se conecta ao servidor de aplicação
b) Cliente se conecta diretamente ao servidor (que inclui banco)
c) Cliente se conecta ao banco de dados
d) Não há servidor

**Resposta:** b) Cliente se conecta diretamente ao servidor (que inclui banco)

---

## Questão 5 (Múltipla Escolha)
Na arquitetura 3-Tier:

a) Cliente se conecta diretamente ao banco
b) Cliente se conecta ao servidor de aplicação, que se conecta ao servidor de BD
c) Cliente e servidor são a mesma coisa
d) Não há servidor de aplicação

**Resposta:** b) Cliente se conecta ao servidor de aplicação, que se conecta ao servidor de BD

---

## Questão 6 (Múltipla Escolha)
Qual é uma vantagem da arquitetura Cliente-Servidor?

a) Centralização de processamento
b) Facilita manutenção
c) Melhora segurança
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 7 (Múltipla Escolha)
Qual arquitetura é mais escalável?

a) 2-Tier
b) 3-Tier
c) São iguais
d) Depende do caso

**Resposta:** b) 3-Tier - permite escalar cada camada independentemente

---

## Questão 8 (Múltipla Escolha)
No fluxo Cliente-Servidor, a sequência é:

a) Cliente → Requisição → Servidor → Processa → Resposta → Cliente
b) Servidor → Cliente → Requisição
c) Cliente → Resposta → Servidor
d) Não há fluxo

**Resposta:** a) Cliente → Requisição → Servidor → Processa → Resposta → Cliente

---

## Questão 9 (Múltipla Escolha)
Na arquitetura 3-Tier, o cliente:

a) Acessa diretamente o banco de dados
b) Não acessa diretamente o banco de dados
c) É o mesmo que servidor
d) Não existe

**Resposta:** b) Não acessa diretamente o banco de dados - passa pelo servidor de aplicação

---

## Questão 10 (Múltipla Escolha)
Qual é uma desvantagem da arquitetura Cliente-Servidor?

a) Servidor pode se tornar gargalo
b) Dependência do servidor
c) Pode ter latência de rede
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima
