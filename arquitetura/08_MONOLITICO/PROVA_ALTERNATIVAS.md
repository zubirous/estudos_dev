# Prova: ARQUITETURA MONOLÍTICA

## Provas de Alternativas

## Questão 1 (Múltipla Escolha)
Qual a principal característica de uma arquitetura monolítica?

a) Múltiplos deploys
b) Um único deploy com todo código
c) Múltiplos bancos de dados
d) Serviços independentes

**Resposta:** b) Um único deploy com todo código

---

## Questão 2 (Múltipla Escolha)
Quando escolher arquitetura monolítica?

a) Sistema muito grande
b) Sistema pequeno/médio, equipe pequena
c) Quando precisa escalar partes específicas
d) Nunca escolher monolítico

**Resposta:** b) Sistema pequeno/médio, equipe pequena

---

## Questão 3 (Múltipla Escolha)
Qual é uma vantagem da arquitetura monolítica?

a) Simplicidade de desenvolvimento
b) Fácil debugging
c) Transações ACID simples
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 4 (Múltipla Escolha)
Qual é uma desvantagem da arquitetura monolítica?

a) Escalabilidade limitada
b) Acoplamento forte
c) Uma falha afeta todo sistema
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 5 (Múltipla Escolha)
Em arquitetura monolítica, quando você precisa fazer uma pequena mudança:

a) Deploy apenas da parte alterada
b) Deploy de todo sistema
c) Não precisa fazer deploy
d) Apenas reiniciar

**Resposta:** b) Deploy de todo sistema - desvantagem dos monolíticos

---

## Questão 6 (Múltipla Escolha)
Arquitetura monolítica compartilha:

a) Banco de dados
b) Código fonte
c) Processo de execução
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima - monolítico é tudo em um
