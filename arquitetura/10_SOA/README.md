# 10. ARQUITETURA SOA (Service-Oriented Architecture)

## Conceito
Sistema composto por serviços reutilizáveis que comunicam via protocolos padronizados.

## Características:
- Serviços reutilizáveis
- Comunicação via protocolos padronizados (SOAP, REST, etc)
- Foco em reutilização de serviços

## Diferença de Microserviços:
- **SOA**: Serviços maiores, mais acoplados, foco em reutilização
- **Microserviços**: Serviços menores, independentes, foco em isolamento

## Quando usar:
- Necessidade de reutilizar serviços entre diferentes aplicações
- Integração com sistemas legados
- Comunicação via protocolos padronizados
