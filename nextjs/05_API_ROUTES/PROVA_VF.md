# Prova: API ROUTES - Next.js

## Provas de Verdadeiro ou Falso

## Questão 1 (Verdadeiro/Falso)
API Routes no Next.js são executadas no servidor.

**Resposta:** Verdadeiro. API Routes são executadas no servidor Node.js, não no cliente.

---

## Questão 2 (Verdadeiro/Falso)
Arquivos em pages/api/ se tornam endpoints de API automaticamente.

**Resposta:** Verdadeiro. Cada arquivo em pages/api/ vira um endpoint em /api/.

---

## Questão 3 (Verdadeiro/Falso)
API Routes podem ter rotas dinâmicas usando [id].js.

**Resposta:** Verdadeiro. Você pode criar rotas dinâmicas como pages/api/user/[id].js.

---

## Questão 4 (Verdadeiro/Falso)
API Routes são similares a Express.js.

**Resposta:** Verdadeiro. O handler recebe req e res, similar ao Express.js.

---

## Questão 5 (Verdadeiro/Falso)
API Routes podem processar apenas requisições GET.

**Resposta:** Falso. API Routes podem processar todos os métodos HTTP (GET, POST, PUT, DELETE, etc).
