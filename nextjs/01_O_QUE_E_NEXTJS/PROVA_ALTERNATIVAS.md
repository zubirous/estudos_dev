# Prova: O QUE É NEXT.JS?

## Provas de Alternativas

## Questão 1 (Múltipla Escolha)
O que é Next.js?

a) Biblioteca JavaScript
b) Framework React para produção
c) Banco de dados
d) Linguagem de programação

**Resposta:** b) Framework React para produção

---

## Questão 2 (Múltipla Escolha)
Qual característica NÃO é do Next.js?

a) Server-Side Rendering
b) File-based Routing
c) API Routes
d) Compilador de C++

**Resposta:** d) Compilador de C++

---

## Questão 3 (Múltipla Escolha)
Qual é a principal diferença entre Next.js e React puro?

a) Next.js adiciona SSR, SSG, roteamento baseado em arquivos
b) React puro é apenas CSR (Client-Side Rendering)
c) Next.js é otimizado para produção
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 4 (Múltipla Escolha)
Next.js foi criado por:

a) Facebook
b) Google
c) Vercel
d) Microsoft

**Resposta:** c) Vercel

---

## Questão 5 (Múltipla Escolha)
Qual característica do Next.js otimiza automaticamente imagens?

a) Image Optimization
b) Code Splitting
c) CSS Support
d) API Routes

**Resposta:** a) Image Optimization - otimização automática de imagens

---

## Questão 6 (Múltipla Escolha)
Qual característica do Next.js divide código automaticamente?

a) Image Optimization
b) Code Splitting
c) SSR
d) API Routes

**Resposta:** b) Code Splitting - divisão automática de código, carregando apenas o necessário

---

## Questão 7 (Múltipla Escolha)
Next.js suporta:

a) Apenas CSR
b) CSR, SSR e SSG
c) Apenas SSR
d) Apenas SSG

**Resposta:** b) CSR, SSR e SSG - Next.js suporta todos os métodos de renderização

---

## Questão 8 (Múltipla Escolha)
Next.js é melhor que React puro para:

a) Aplicações que precisam de SEO
b) Aplicações que precisam de SSR/SSG
c) Aplicações que precisam de otimizações automáticas
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 9 (Múltipla Escolha)
O que Next.js adiciona ao React?

a) Apenas roteamento
b) SSR, SSG, API Routes, roteamento baseado em arquivos, otimizações
c) Apenas API Routes
d) Nada

**Resposta:** b) SSR, SSG, API Routes, roteamento baseado em arquivos, otimizações automáticas

---

## Questão 10 (Múltipla Escolha)
Qual plataforma criou o Next.js?

a) Facebook
b) Google
c) Vercel
d) Microsoft

**Resposta:** c) Vercel - Next.js foi criado e é mantido pela Vercel
