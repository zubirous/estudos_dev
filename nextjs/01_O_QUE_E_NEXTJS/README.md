# 1. O QUE É NEXT.JS?

Next.js é um **framework React** para produção, criado pela **Vercel**.

## Principais Características:
- **Server-Side Rendering (SSR)**: Renderização no servidor
- **Static Site Generation (SSG)**: Geração de sites estáticos
- **API Routes**: Criar APIs dentro do projeto
- **File-based Routing**: Roteamento baseado em arquivos
- **Automatic Code Splitting**: Divisão automática de código
- **Built-in CSS Support**: Suporte a CSS integrado
- **Image Optimization**: Otimização automática de imagens

## Diferenças do React puro:

- **Next.js adiciona**: SSR, SSG, API Routes, roteamento baseado em arquivos, otimizações automáticas
- **React puro é apenas CSR** (Client-Side Rendering)
- **Next.js é otimizado para produção**

## Métodos de Renderização:

**Next.js suporta:**
- ✅ **CSR** (Client-Side Rendering)
- ✅ **SSR** (Server-Side Rendering)
- ✅ **SSG** (Static Site Generation)

## Quando usar Next.js ao invés de React puro:

- ✅ Aplicações que precisam de **SEO**
- ✅ Aplicações que precisam de **SSR/SSG**
- ✅ Aplicações que precisam de **otimizações automáticas**
- ✅ Aplicações que precisam de **API Routes** (sem servidor separado)
