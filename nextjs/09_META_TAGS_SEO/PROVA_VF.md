# Prova: META TAGS E SEO - Next.js

## Provas de Verdadeiro ou Falso

## Questão 1 (Verdadeiro/Falso)
O componente Head do Next.js é usado para definir meta tags.

**Resposta:** Verdadeiro. O componente Head do next/head é usado para definir meta tags, título e outras tags do <head>.

---

## Questão 2 (Verdadeiro/Falso)
Meta tags são importantes para SEO.

**Resposta:** Verdadeiro. Meta tags como title, description e keywords são importantes para SEO e aparecem nos resultados de busca.

---

## Questão 3 (Verdadeiro/Falso)
Open Graph tags são usadas para redes sociais.

**Resposta:** Verdadeiro. Open Graph tags (og:title, og:description, etc) são usadas para definir como o conteúdo aparece quando compartilhado em redes sociais.

---

## Questão 4 (Verdadeiro/Falso)
Cada página pode ter suas próprias meta tags no Next.js.

**Resposta:** Verdadeiro. Cada página pode definir suas próprias meta tags usando o componente Head, permitindo personalização por página.

---

## Questão 5 (Verdadeiro/Falso)
O componente Head deve ser importado de 'next/head'.

**Resposta:** Verdadeiro. O componente Head é importado de 'next/head' para definir meta tags.

---

## Questão 6 (Verdadeiro/Falso)
Meta tags são visíveis na página renderizada.

**Resposta:** Falso. Meta tags aparecem no código HTML (<head>), mas não são visíveis na página renderizada. Elas são usadas por mecanismos de busca e navegadores.

---

## Questão 7 (Verdadeiro/Falso)
Open Graph tags melhoram a aparência de links compartilhados em redes sociais.

**Resposta:** Verdadeiro. Open Graph tags definem título, descrição, imagem e outros dados que aparecem quando um link é compartilhado em redes sociais como Facebook, Twitter, etc.

---

## Questão 8 (Verdadeiro/Falso)
Meta tags são apenas para SEO.

**Resposta:** Falso. Meta tags são usadas para SEO, mas também para redes sociais (Open Graph), navegadores (viewport, charset), e outras funcionalidades.
