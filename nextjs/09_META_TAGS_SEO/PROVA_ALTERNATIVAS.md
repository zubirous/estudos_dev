# Prova: META TAGS E SEO - Next.js

## Provas de Alternativas

## Questão 1 (Múltipla Escolha)
Qual componente do Next.js é usado para meta tags?

a) <Meta>
b) <Head> do next/head
c) <Title>
d) <Tags>

**Resposta:** b) <Head> do next/head

---

## Questão 2 (Múltipla Escolha)
Para que servem as meta tags no Next.js?

a) Apenas para SEO
b) SEO, redes sociais, navegadores
c) Apenas para redes sociais
d) Apenas para navegadores

**Resposta:** b) SEO, redes sociais, navegadores

---

## Questão 3 (Múltipla Escolha)
O que são Open Graph tags?

a) Tags para SEO
b) Tags para definir como conteúdo aparece em redes sociais
c) Tags para navegadores
d) Tags para banco de dados

**Resposta:** b) Tags para definir como conteúdo aparece em redes sociais

---

## Questão 4 (Múltipla Escolha)
Como definir o título da página no Next.js?

a) <title>Minha Página</title>
b) <Head><title>Minha Página</title></Head>
c) <Title>Minha Página</Title>
d) Não é possível

**Resposta:** b) <Head><title>Minha Página</title></Head>

---

## Questão 5 (Múltipla Escolha)
Como definir meta description no Next.js?

a) <meta description="...">
b) <Head><meta name="description" content="..." /></Head>
c) <Description>...</Description>
d) Não é possível

**Resposta:** b) <Head><meta name="description" content="..." /></Head>

---

## Questão 6 (Múltipla Escolha)
O que são meta tags?

a) Tags visíveis na página
b) Tags no <head> do HTML, não visíveis na página
c) Tags apenas para JavaScript
d) Tags apenas para CSS

**Resposta:** b) Tags no <head> do HTML, não visíveis na página, mas usadas por mecanismos de busca e navegadores

---

## Questão 7 (Múltipla Escolha)
Cada página no Next.js pode ter:

a) Apenas as mesmas meta tags
b) Suas próprias meta tags personalizadas
c) Apenas título
d) Nenhuma meta tag

**Resposta:** b) Suas próprias meta tags personalizadas - cada página pode definir suas próprias meta tags

---

## Questão 8 (Múltipla Escolha)
Open Graph tags começam com:

a) og:
b) meta:
c) head:
d) seo:

**Resposta:** a) og: - exemplo: og:title, og:description, og:image

---

## Questão 9 (Múltipla Escolha)
Meta tags são importantes para:

a) Apenas SEO
b) SEO e redes sociais
c) Apenas redes sociais
d) Nada

**Resposta:** b) SEO e redes sociais - melhoram resultados de busca e aparência em redes sociais

---

## Questão 10 (Múltipla Escolha)
Como importar o componente Head no Next.js?

a) import Head from 'next/head'
b) import Head from 'next'
c) import { Head } from 'react'
d) Não precisa importar

**Resposta:** a) import Head from 'next/head'
