# Prova: CSS E ESTILOS - Next.js

## Provas de Verdadeiro ou Falso

## Questão 1 (Verdadeiro/Falso)
CSS global deve ser importado no _app.js no Next.js.

**Resposta:** Verdadeiro. CSS global deve ser importado no _app.js para ser aplicado em todas as páginas.

---

## Questão 2 (Verdadeiro/Falso)
CSS Modules no Next.js usam extensão .module.css.

**Resposta:** Verdadeiro. CSS Modules no Next.js usam extensão .module.css e são importados como objetos.

---

## Questão 3 (Verdadeiro/Falso)
CSS Modules garantem que estilos não vazem para outros componentes.

**Resposta:** Verdadeiro. CSS Modules geram nomes de classes únicos, garantindo que estilos não vazem para outros componentes.

---

## Questão 4 (Verdadeiro/Falso)
Styled JSX é built-in no Next.js.

**Resposta:** Verdadeiro. Styled JSX vem built-in no Next.js, permitindo escrever CSS dentro de componentes.

---

## Questão 5 (Verdadeiro/Falso)
CSS Modules podem ser usados em qualquer componente do Next.js.

**Resposta:** Verdadeiro. CSS Modules podem ser importados e usados em qualquer componente do Next.js.

---

## Questão 6 (Verdadeiro/Falso)
CSS global pode ser importado em qualquer componente.

**Resposta:** Falso. CSS global deve ser importado apenas no _app.js para evitar duplicação e problemas de ordem de importação.

---

## Questão 7 (Verdadeiro/Falso)
Styled JSX permite escrever CSS dentro de componentes React.

**Resposta:** Verdadeiro. Styled JSX permite escrever CSS dentro de componentes usando a tag <style jsx>.

---

## Questão 8 (Verdadeiro/Falso)
CSS Modules são úteis para estilos compartilhados entre componentes.

**Resposta:** Falso. CSS Modules são úteis para estilos escopados por componente. Para estilos compartilhados, use CSS global ou Styled JSX.

---

## Questão 9 (Verdadeiro/Falso)
No Next.js, você pode usar apenas uma forma de estilização (CSS Modules, Styled JSX, ou CSS global).

**Resposta:** Falso. Você pode usar CSS Modules, Styled JSX e CSS global juntos no mesmo projeto Next.js.

---

## Questão 10 (Verdadeiro/Falso)
CSS Modules no Next.js são compilados para nomes de classes únicos.

**Resposta:** Verdadeiro. CSS Modules são compilados para gerar nomes de classes únicos, evitando conflitos de estilos.
