# Prova: STORED PROCEDURES - MySQL

## Provas de Verdadeiro ou Falso

## Questão 1 (Verdadeiro/Falso)
Stored Procedures são executadas no servidor do banco de dados.

**Resposta:** Verdadeiro. Stored Procedures são executadas no servidor, não no cliente.

---

## Questão 2 (Verdadeiro/Falso)
Stored Procedures podem receber parâmetros.

**Resposta:** Verdadeiro. Stored Procedures podem receber parâmetros IN, OUT e INOUT.

---

## Questão 3 (Verdadeiro/Falso)
Stored Procedures podem executar lógica SQL complexa.

**Resposta:** Verdadeiro. Stored Procedures podem conter lógica SQL complexa com loops, condições, etc.

---

## Questão 4 (Verdadeiro/Falso)
Para criar uma stored procedure, é necessário alterar o DELIMITER.

**Resposta:** Verdadeiro. Geralmente é necessário alterar o DELIMITER para // ou $$ para permitir múltiplos comandos dentro da procedure.

---

## Questão 5 (Verdadeiro/Falso)
Stored Procedures são reutilizáveis.

**Resposta:** Verdadeiro. Uma vez criada, uma stored procedure pode ser chamada múltiplas vezes, sendo reutilizável.

---

## Questão 6 (Verdadeiro/Falso)
Stored Procedures executam no servidor do banco de dados, não no cliente.

**Resposta:** Verdadeiro. Stored Procedures são executadas no servidor, reduzindo latência de rede e melhorando performance.

---

## Questão 7 (Verdadeiro/Falso)
É necessário alterar o DELIMITER para criar stored procedures no MySQL.

**Resposta:** Verdadeiro. Geralmente é necessário alterar o DELIMITER (para // ou $$) para permitir múltiplos comandos dentro da procedure.

---

## Questão 8 (Verdadeiro/Falso)
Stored Procedures podem retornar múltiplos resultados.

**Resposta:** Verdadeiro. Stored Procedures podem retornar múltiplos resultados usando múltiplos SELECT statements.

---

## Questão 9 (Verdadeiro/Falso)
Stored Procedures podem ter parâmetros de entrada, saída e entrada/saída.

**Resposta:** Verdadeiro. Stored Procedures podem ter parâmetros IN (entrada), OUT (saída) e INOUT (entrada/saída).
