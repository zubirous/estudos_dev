# Prova: STORED PROCEDURES - MySQL

## Provas de Alternativas

## Questão 1 (Múltipla Escolha)
Como chamar uma stored procedure chamada "obter_usuario"?

a) SELECT obter_usuario(1)
b) CALL obter_usuario(1)
c) EXEC obter_usuario(1)
d) RUN obter_usuario(1)

**Resposta:** b) CALL obter_usuario(1)

---

## Questão 2 (Múltipla Escolha)
O que são Stored Procedures?

a) Tabelas armazenadas
b) Funções SQL armazenadas no banco que podem receber parâmetros
c) Índices especiais
d) Constraints avançadas

**Resposta:** b) Funções SQL armazenadas no banco que podem receber parâmetros

---

## Questão 3 (Múltipla Escolha)
Para criar uma stored procedure, é necessário usar:

a) BEGIN e END
b) START e STOP
c) OPEN e CLOSE
d) CREATE e DROP

**Resposta:** a) BEGIN e END - delimitam o bloco de código da procedure

---

## Questão 4 (Múltipla Escolha)
Qual comando é usado para deletar uma stored procedure?

a) DELETE PROCEDURE
b) DROP PROCEDURE
c) REMOVE PROCEDURE
d) DROP FUNCTION

**Resposta:** b) DROP PROCEDURE

---

## Questão 5 (Múltipla Escolha)
Stored Procedures podem receber parâmetros?

a) Não, nunca
b) Sim, podem receber parâmetros de entrada (IN)
c) Apenas parâmetros de saída
d) Apenas um parâmetro

**Resposta:** b) Sim, podem receber parâmetros de entrada (IN), saída (OUT) e entrada/saída (INOUT)

---

## Questão 6 (Múltipla Escolha)
Onde stored procedures são executadas?

a) No cliente
b) No servidor do banco de dados
c) No navegador
d) Não são executadas

**Resposta:** b) No servidor do banco de dados - procedures são executadas no servidor

---

## Questão 7 (Múltipla Escolha)
Qual é a sintaxe para chamar uma stored procedure?

a) SELECT procedure_name()
b) CALL procedure_name()
c) EXEC procedure_name()
d) RUN procedure_name()

**Resposta:** b) CALL procedure_name() - comando CALL é usado para executar stored procedures

---

## Questão 8 (Múltipla Escolha)
Por que usar stored procedures?

a) Executam no servidor (mais rápido)
b) Podem ser reutilizadas
c) Podem executar lógica SQL complexa
d) Todas as alternativas acima

**Resposta:** d) Todas as alternativas acima

---

## Questão 9 (Múltipla Escolha)
Qual comando é usado para deletar uma stored procedure?

a) DELETE PROCEDURE
b) DROP PROCEDURE
c) REMOVE PROCEDURE
d) DROP FUNCTION

**Resposta:** b) DROP PROCEDURE - comando para deletar stored procedures

---

## Questão 10 (Múltipla Escolha)
Stored procedures podem conter:

a) Apenas SELECT
b) Lógica SQL complexa com loops, condições, etc
c) Apenas INSERT
d) Apenas UPDATE

**Resposta:** b) Lógica SQL complexa com loops, condições, etc - procedures podem conter lógica complexa
